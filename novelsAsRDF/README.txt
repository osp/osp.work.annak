Ce dossier collecte des interpretations des récits d'Anna Kavan sous la forme
de descriptions RDF (au format
[Turtle](http://fr.wikipedia.org/wiki/Turtle_(syntaxe))) 

Pour ajouter les déclarations RDF à la base de données RDF, lancer la commande
`rdfproc -c store parse fichier.turtle turtle` (où `store` est le nom de la
base de données).
